The Horror of Images

I made a site that randomly gives you images it finds on Imgur. LINK

It works by generating a random string, appending it to an Imgur URL and fetching the image - there are a couple of 404 checks in there to try and detect when it fails and re-roll for a new image, but I'm still trying to work out all the edge cases where Imgur doesn't return a valid image. As a result it's a bit broken but still fun to muck about with.
